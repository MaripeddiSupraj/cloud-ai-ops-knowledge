# Cloud AI Ops Knowledge

Cloud AI Ops Knowledge is the public reading repository for curated DevOps, cloud, and AI notes.

## What lives here

- readable markdown entries in `content/`
- browse indexes in `indexes/`
- lightweight metadata in `registry.json`

## What does not live here

- ingestion code
- classification logic
- enhancement pipeline
- publishing automation

That logic now lives in the separate core repo:

- [cloud-ai-ops-core](https://github.com/MaripeddiSupraj/cloud-ai-ops-core)

## Browse

- `indexes/all.md`
- `indexes/by-category.md`
- `indexes/by-tool.md`
- `indexes/by-content-type.md`

This repo is intentionally kept simple so it works as a clean public knowledge library.
